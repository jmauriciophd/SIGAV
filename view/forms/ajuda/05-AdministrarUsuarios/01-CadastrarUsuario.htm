Cadastrar Usuário

O Cadastro de Usuário é um pequeno formulário que é usado para cadastrar um usuário e definindo perfil para acesso ao SIGAV.

Todos os campos são de preenchimento obrigatórios tendo como campos de informações:

CPF, Nome, e-mail,senha, confirma senha e perfil.

CPF é o campo de preenchimento do CPF do usuário.;

Nome é o campo de preenchimento do nome do usuário;

E-mail é ocampo de preenchimentodo e-mailprofissional do usuário;

Senha é o campo de cadastro de senha do usuário;

Confirmar Senhaéo campoonde o usuário repete asenhapara confirmar a senha de acessoao SIGAV.

Perfil é o campo onde o usuário escolhe o perfil já pré cadastradono sistema.

Após o preenchimento do formulário o usuário decide cadastrar para confirmar o cadastro do usuário ou cancela votando para a tela inicial.
