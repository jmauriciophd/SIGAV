Alugar Vestuário

 

É um formulário que serve para informar e preencher os dados necessários para aluguel das peças de vestuários da loja Jovem Maria.

Na parte de funcionário é a informação do usuário está logado aparecendo o nome do funcionário e data/hora do aluguel, para informação do próprio sistema.

Dentro desse formulárioo cliente já pré cadastrado informa somente o CPF, com o campo de preenchimento obrigatório, aparecerá os dados desse cliente.

Logo após o usuário preenche os dados de Vestuário com o código da peça no campo códgo clicando em adicionar e na lista abaixo será adicionado a peça selecionada pelo código, podendo adicionar quantos itens o cliente for alugar.

O botão Remover serve para excluir uma peça selecionada da lista de aluguel das peças a serem alugadas, fazendo com que a peça não seja mais alugada.

Abaixo da lista de aluguel existe a pegunta: "Deseja agendar prévia?", marcando sim será aberto uma aba abaixo chamaado de selecione uma data para o cliente escolher uma data desejada para a disponibilização da(s) peça(s) a ser alugada(s).

Senão a seleção for não a peça será alugada na hora do preenchimento desse formulário.

Abaixo desse campo tem os dois botões pagar que será encaminhado ao formulário de pagamento de aluguel, ou cancelar, para cancelar o procedimento de aluguel de vestuário.
