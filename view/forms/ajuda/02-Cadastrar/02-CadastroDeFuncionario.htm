Cadastro de Funcionário

Obs: é necessário observar os campos de preenchimento obrigatório marcado com asterístico.

O cadastro de funcionário é um formulário do sistema SIGAV onde é feito o cadastro do funcionário de todos seus dados disponíveis que a loja precisa para obter um controle de quem trabalha na loja Jovem Maria.

O formulário é dividido em quatro sub formulários: Dados pessoais, Dados de endereço, Dados de contato e dados complementares.


Dados pesoais:

Foto 3x4: O usuário escolhe uma foto armazenada no computador, por isso é preciso que use uma foto do tipo arquivo de resolução de média resolução tirada do próprio computador por uma webcam.Senão pode ser transferida de outras formas, desde que não seja uma foto que não dê para armazenar no espaço desse formulário.

Nome: O nome é o do funcionário completo sendo um campo de preenchimento obrigatório.

RG: É o campo de preenchimento de RG do funcionário sendo um campo de preenchimento obrigatório.

Estado Civil: Nesse campo o usuário pode selecionar o estado civil do funcionário sendo um campo de preenchimento obrigatório.

Sexo: Existe duas opções para preenchimento desse item (masculino ou feminino).

CPF: É o campo onde é preenchido o CPF do funcionário onde um campo de preenchimento obrigatório.

Órgão Expedidor: É o campo do órgão onde o RG foi expedido junto com o Estado de expedição onde um campo de preenchimento obrigatório.

Data de Nascimento: É o campo de preenchimento da data de nascimento do funcionário onde um campo de preenchimento obrigatório.

Grau de Instrução: É o campo que o funcionário informa qual a situação educacional dividido pelas seguintes situações:

Ensino fundamental completo, Ensino fundamental incompleto, Ensino médio completo, Ensino médio incompleto, Ensino superior completo, Ensino superior incompleto, Ensino Profissionalizante/Técnico.


Dados do Endereço:

CEP: É o numero do CEP de onde o funcionário reside.Na lupa ao lado é possível achar a quadra e cidade de onde o funcionário reside.

Endereço: É o campo do endereço do funcionário que pode ser achado pelo CEP ou preenchido manualmente.

Cidade: É o campo da cidade de onde é o endereço de onde o funcionário mora.

Número: Número da casa do endereço fornecido.

Estado: É o campo que o funcionário informa escolhendo de acordo com o endereço fornecido.

Bairro: É o campo que do Bairro onde está situado o endereço do funcionário.

Complemento: Esse campo complementa algum dado do endereço do funcionário quando for necessário.


Dados do Contato:

E-mail: É o campo que o funcionário informa seu e-mail para contatos de e-mail profissional.

Telefone Celular: É o campo que o funcionário fornece seu número de celular.

Telefone residencial: É o campo que o funcionário fornece o numero de telefone de sua residência.

Telefone Celular 2: É o campo que o funcionário fornece seu número de celular se houver mais de um número de aparelho celular.


Dados complementares:

Nº.CTPS: É o campo que o funcionário fornece o número da sua carteira de trabalho.

Nº Série: É o campo que o funcionário fornece do número de série da sua carteira de trabalho.

Cargo: É o campo que é preenchido no cargo que o funcionário irá exercer na Jovem Maria.

Salário: É o campo que é preenchido com o salário que será pago pela Jovem Maria.

Comissão: é o campo que é calculado a comissão do funcionário adicional do salário do mesmo.

Data de Admissão: É o campo da data que o funcionário começou a trabalhar na ljoa Jovem Maria.

Deseja criar um usuário para este funcionário? : É a decisão se o usuário terá acessibilidade das funções do SIGAV.Após esse procedimento o administrador poderá personalizar os acessos do SIGAV para este usuário criado.

Botões "Cadastrar" e "Cancelar": São botões para confirmar ou cancelar o cadastro de funcionário por decisão de quem preenche.
