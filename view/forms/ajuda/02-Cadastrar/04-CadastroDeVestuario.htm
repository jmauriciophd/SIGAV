Cadastro de Vestuário:

O cadastro de Vestuário é um formulário do Sistema SIGAV que tem a função de cadastrar as peças de vestuário da loja Jovem Maria para ter controle do que tem a disposição de aluguel desses vestuários e informar as caracteríticas das peças de vestuário.

No formulário há campos de preenchimento dos dados do vestuário que são:

Categoria:Nesse campo o usuário escolhe a categoria préviamente cadastrada em (Cadastrar Categoria) escolhe um dos itens.

Nome: É o campo que o usuário nomeia o vestuário.

Tamanho: É o campo que o usuário numera o tamanho do vestuário. Ex: P, M, G ...

Medidas: É o campo que o usuário numera o tamanho do vestuário.Ex: 38, 40, 42, etc.

Valor do Vestuário: É o campo que o usuário declara o valor total do vestuário em Reias até mesmo para se basear no valor do aluguel.

Valor do Aluguel: É o campo que o usuário declara o valor a ser alugado do vestuário.

Descrição: É o campo que o usuário descreve as características estéticas do vestuário ou particularidades do mesmo.

Observação: É o campo que o usuário pode usar para fazer alguma observação em relação ao vestuário de referência do formulário.

Botões "Cadastrar" e "Cancelar": São botões para confirmar ou cancelar o cadastro de funcionário por decisão de quem preenche.

Obs: é necessário observar os campos de preenchimento obrigatório marcado com asterístico.
