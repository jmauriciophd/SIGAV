Cadastro de Cliente

O cadastro de cliente é um formulário do sistema SIGAV que cadastra novos clientes para melhor controle e relacionamento com o cliente para fins de manipulação dos dados e informações do cliente e manipulação do próprio sistema.

O cadastro é feito em três sub-divisões: Dados pessoais, dados de endereço e dados de contato.

Nos Dados pessoais tem os campos de cpf, sendo preenchido apenas com o númeo de CPF do cliente com preenchimento obrigatório.

Campo do RG parecido com o do CPF é preenchido somente com o número do RG do Cliente e campo obrigatória de preenchimento.

No campo Órgão Expedidor/UF é preenchido órgão de onde é expedido o RG do cliente e a Unidade da Federação existe as opções dos estados de origem do RG.

Data de Nascimento é preenchido somente com número e é campo de preenchimeto obrigatório.

O campo Nome é de preenchimento obrigatório do nome do Cliente.

No campo de Estado Civil já existe a opção da situção do estado civil do cliente: Solteiro(a), Casado(a), Divorciado(a) e Viúvo(a).

No campo sexo o cliente marca seu genero de sexo: masculino ou feminino.

Dados do Endereço é onde o cliente cadastra sua residência.

No campo CEP o preenchimento desse item já mostra a quadra ou rua de onde é cadastrado o CEP do local, nesse preenchimento é colocado o cEP e depois clica na lupa ao lado e é feito a pesquisa e preenhimento automático.Senão o preenchimento pode ser manual, ou seja, em cada campo é preenchido de acordo com o que se pede.

No campo Endereço o cliente fornece seu endereço de residência sendo um campo de preenchimento obrgatório.

No campo bairro é colocado o Bairro da residência onde se localiza a residência do cliente sendo um campo de preenchimento obrgatório.

No campo Cidade é preenchido com o nome da cidade do enderreço do cliente sendo um campo de preenchimento obrgatório.

No campo Estado é mostrado as opções dos estados do Brasil com a escolha do mesmo de onde se localiza a residência do cliente sendo um campo de preenchimento obrgatório.

O complemento se for necessário para terminar essa parte dos dados de endereço.

Dados do contato é a parte final do cadastro de contato é para obter o maior número de informações de contato do cliente para melhor aproximação do cliente a loja.

O espaço do e-mail é fornecido um e-mail de contato para enviar prováveis informações como preços reservas ou algo do tipo.

Tel. Residencial onde é preenchido o número de telefone de sua residência.

Telefone celular onde o cliente cadastra o número de seu número de celular sendo um campo de preenchimento obrgatório.

Twitter, Facebook e orkut se houver e o cliente quiser disponibilizar poderá obter para um relacionamento melhor com o cliente.

No rodapé do formulário tem os dois botões de cadastrar que depois de preenchido todo o formulário clique e será cadastrado, nesse momento vai aparecer na parte superior do a mensagem "O cliente foi cadastrado com sucesso.". E no botão cancelar se clicar será cancelada o cadastro do cliente.
