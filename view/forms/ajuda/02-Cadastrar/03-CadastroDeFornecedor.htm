Cadastro de Fornecedor:

O cadastro de fornecedor é um formulário que cadastra os fornecedores para loja Jovem Maria para fins de melhorar as relações comerciais entre fornecedor e a própria loja e manter mais próxima do contato quando for necessário.Abaixo as explicações para melhor enteder o formulário em caso de dúvidas de preenchimento.


Dados Básicos:

CNPJ: É o campo que o usuário preenche com o número do CNPJ do fornecedor.

Razão Social: É o campo que o usuário preenche com o nome de Razão social cadastrado em cartório, ou nome oficial da empresa fornecedora.

Nome Fantasia: É o campo que o usuário preenche com o nome fantasia, nome que o fornecedor usa para fins comerciais de sua empresa.

Inscrição Estadual: É o campo que o usuário preenche com é o registro do contribuinte no cadastro do ICMS mantido pela Receita Estadual. Com a inscrição, o contribuinte do ICMS passa a ter o registro formal do seu negócio, junto à Receita Estadual do estado onde está estabelecido.


Dados do Endereço:

CEP: É o campo que o usuário preenche com o número do CEP do fornecedor; na lupa ao lado é capaz que o SIGAV encontre o endereço com o preenchimento desse campo.

Endereço: É o campo do endereço do Fornecedor que pode ser achado pelo CEP ou preenchido manualmente.

Bairro: É o campo do Bairro onde está situado o endereço do Fornecedor.

Estado: É o campo que o Fornecedor informa escolhendo de acordo com o endereço fornecido.

Número: Número do local do endereço fornecido pelo Fornecedor.

Cidade: É o campo preenchido da cidade de onde é o endereço de onde o Fornecedor situa.

Complemento: Esse campo complementa algum dado do endereço do Fornacedor quando for necessário.


Dados do contato:

E-mail: É o campo que o Fornecedor informa seu e-mail para contatos de e-mail profissional.

Telefone Celular1: É o campo que o Fornecedor mantém seu número de celular.

Telefone Comercial: É o campo que o Fornecedor mantém seu número de telefone comercial para contatos e negócios.

Telefone Celular2: É o campo que o Fornecedor mantém seu número de outro celular quando houver necessidade de maior contato com a Jovem Maria.

Botões "Cadastrar" e "Cancelar": São botões para confirmar ou cancelar o cadastro de Fornecedor por decisão de quem preenche.
